# 🕹️ Avery's Arcade

A playful, colorful mini-game arcade with 3 games.

## 🎮 Games

### 🏐 Spike Star Rally
Rally against an AI opponent on 5 fantasy courts (Cloud, Underwater, Jungle, Candy, Volcano). Choose Easy or Challenge difficulty, pick your side, and play first to 25 points. Spike the ball by jumping and hitting!

### ⭐ Volley Quest
A Mario-style side-scroller with a volleyball theme. 3 levels (Practice Court, Beach Tournament, Championship Arena). Jump on rogue volleyballs and dodge referee whistles. Collect bronze, silver, and gold medals. Find the hidden bonus volleyball in each level. End every level with a big spike at the net!

### 🎨 Doodle Party
A creative drawing canvas with fun prompts, 12 colors, 4 brush sizes, and an eraser. Save your art to earn coins!

## ✨ Features

- Avatar customization (color, face, flair)
- Coin economy earned through gameplay
- Unlock shop with characters, themes, and accessories
- High score leaderboard
- All data saved locally in the browser

## 🚀 Quick Start (No Install)

Just open `public/index.html` in any browser. That's it!

## 🌐 Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Set **Output Directory** to `public`
4. Click Deploy → Done!
